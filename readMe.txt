02. Recommendations for proper usage:

    - It is important to state that the main file for the webpage is located at the "Pages" folder
    and referenced as "index.html";

    - In addition, if you have any problem running the website through its files, it should probably
    be related with the relative paths for each component of the website;

    - If that happens, it is included a .zip file without folders and paths, look for the
    "index.html" file and run it;

    - Finally, make sure to use a live preview extension/tool and to verify if your browser of choice is
    properly updated.

    - Here's a cool tip of usage for the home page: click once to expand the images and twice to collapse them back!

    Thank you! :)
